ZTNA

Private Access

Zero Trust Network Access

VPN을 대체하여 네트워크 전체가 아닌 특정 애플리케이션에만 1:1로 연결합니다. 내부망의 불필요한 노출을 제거합니다.

SWG

Web Security

Secure Web Gateway

웹 트래픽을 실시간으로 분석하여 악성코드 차단 및 유해 사이트 접속을 제어합니다. 안전한 인터넷 브라우징을 보장합니다.

CASB

SaaS Control

Cloud Access Security Broker

SaaS 애플리케이션(Slack, Google Workspace 등)에 대한 가시성 확보 및 세분화된 접근 제어 정책을 적용합니다.

DLP

Data Protection

Data Loss Prevention

웹, 클라우드, 이메일 등 모든 경로에서의 중요 데이터 유출을 식별하고 차단합니다. 규제 준수(Compliance)를 지원합니다.
